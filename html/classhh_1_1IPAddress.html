◆ Create()

IPAddress::ptr hh::IPAddress::Create ( const char *  address,
uint16_t  port = 0 
)
static

根据给定的地址和端口创建一个IPAddress对象的智能指针。

参数
address字符串形式的IP地址，支持IPv4和IPv6。
port端口号，使用网络字节序。
返回
返回一个IPAddress对象的智能指针，如果创建失败则返回nullptr。

本函数首先使用getaddrinfo函数解析地址字符串，然后尝试创建一个IPAddress对象。 如果解析地址失败，函数将记录错误信息并返回nullptr。 成功解析地址后，函数将释放相关的addrinfo结构体。
